## Sistema de Pedidos de Camisetas

Este programa permite que o usuário escolha o modelo de camiseta, a quantidade desejada e o tipo de frete, calculando automaticamente o valor total do pedido com descontos aplicáveis.

## Funcionalidades
- Seleção do modelo de camiseta (MCS, MLS, MCE, MLE)
- Definição da quantidade de camisetas
- Cálculo automático do valor bruto e valor com desconto
- Escolha do tipo de frete (fábrica, transportadora, Sedex)
- Exibição do valor total do pedido

## Tecnologias
- Python 3

## Como executar
```bash
python pedido_camisetas.py
